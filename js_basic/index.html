js语法基础

发表于2021-06-30|更新于2021-07-02|js基础

基础规则

  1. 区分大小写

  2. 标识符, 所谓的标识符就是变量, 函数, 属性或者函数参数的名称; 标识符可以由一个或者多个下列字符组成:

    1. 第一个字符必须是字母, 下划线或者$美元符
    2. 剩下的其他字符可以是字母, 下划线, $美元符或者数字
    3. 标识符中的字母可是 ascii 中的字母也可以是 unicode 中的字母字符, 但是不推荐使用 unicode;
    4. 按照惯例, 标识符使用驼峰大小写来命名; 第一个单词的首字母小写, 后面的单词的首字母大写;

  3. 注释:

    1. 单行注释: // 单行注释
    2. 多行注释: /* 多行注释 */

  4. 严格模式: es6 中增加了严格模式 strict mode 模式, 严格模式中会将 es3 中一些不规范的写法进行规范, 抛出错误;

    1. 对整个脚本开启严格模式, 在脚本开头增加 "use strict";

    2. 对某个函数增加严格模式, 在函数开头加上预处理指令

  5. es 中语句默认使用 ; 结尾, 如果省略了分号, 那么解释器在解释代码的过程中会自动加上分号; 推荐每个语句结束的时候手动加上分号, 防止错误

关键字和保留字

保留的关键字不能作为标识符或属性名

关键词
break do in typeof
case else instanceof var
catch export new void
class extends return while
const finally super with
continue for switch yield
debugger function this
default if throw
delete import try
始终保留
enum
严格模式下保留
implements package public interface
protected static let private
模块代码中保留
await
