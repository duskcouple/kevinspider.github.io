变量

发表于2021-06-30|更新于2021-07-08|js基础

变量

es 中变量是松散的, 变量可以用于保存任何类型的数据;

变量名知识用于保存任意变量而存在的占位符;

有三个关键字可以声明变量, var, let 和 const; 其中 var 可以在 es 任意版本中使用, let 和 const 只能在 es6 以及更晚的版本中使用;

var 声明

要定义变量, 可以使用 var 操作符, 后面跟上变量名; var message; 这行代码定义了一个名为 message的变量, 可以用它来存放任意的数据类型; 在不进行初始化的情况下, 默认会给变量赋值 undefined

作用域

var 声明的作用域会作用于包含它的函数内部, 作为局部变量; 在函数退出的时候进行销毁;

去掉 var 进行声明的变量就是全局变量, 全局变量可以在所有函数的作用域中使用;

全局提升为 window 属性

声明多个变量

需要定义多个变量, 可以在一条语句中用逗号进行分隔;

var 声明提升

let 声明

作用域

let 和 var 的作用差不多, 但是最明显的区别就是 let 的作用域是块, 而 var 的作用域是函数;

块作用域是函数作用域的子集;

不允许多次声明

使用 let 声明变量, 不允许在块作用域中出现多次的冗余声明; var 则支持多次声明;

在 html 中通过 let 声明相同的变量也会报错

不支持变量提升

let 和 var 另一个重要区别是 let 不支持变量提升;

let 不会成为 window 属性

循环中使用 let

const 声明

const 的行为和 let 基本相同, 都是作用于块作用域中, 唯一的区别就是 const 声明的变量必须同时初始化, 且不允许改变;

不允许修改 const 的值

const 不允许重复声明

const 不能用于循环计数

const 可以用于 for-in 和 for-of

在 for-of 和 for-in 中, 使用 const 相当于每次都创建一个新的变量;

使用建议

  1. 尽量不使用 var 进行变量的定义, 有了 let 和 const 之后, 大多数情况下都不需要使用 var 了, 强制自己使用 let 和 const, 提高代码质量
  2. 优先使用 const, 只有在提前预知未来会修改该变量的值时才使用 let;
