数据类型

发表于2021-06-30|更新于2021-07-08|js基础

数据类型

在 js 中所有的值都可以用下列的 7 种数据类型进行表示;

es 中有 6 种原始数据类型:

  1. Undefined
  2. Null
  3. Boolean
  4. Number
  5. String
  6. Symbol (es6 中新增)

es 中有 1 种复杂的数据类型:

  1. Object: 一种无序的键值对的集合;

typeOf 操作符

typeOf 操作符是用来确定任意数据的数据类型, 对任意一个数据使用操作符返回的结果为:

  1. “undefined”: 未定义
  2. “boolean”: bool 类型
  3. “string”: 字符串
  4. “number”: 数值
  5. “object”: 对象(而不是函数) 或者 null;
  6. “function”: 函数
  7. “symbol” : 符号

Undefined 类型

Undefined 类型只有一个值, 就是特殊值 undefined; 当使用 var 或者 let 对变量进行声明的时候, 如果没有对变量进行赋值, 那么变量默认就是undefined

undefined 和 null 可以区分一个变量是一个空指针对象(null) 还是一个未初始化的变量; null –> 空指针对象 –> Obejct; undefined –> 未初始化的变量;

Null 类型

Null 类型同样只有一个值, 就是特殊值 null; 逻辑上讲, null 值表示一个空指针对象, 所以对一个 null 变量使用 typeOf 操作符会返回一个 object;

在声明一个未来要保存对象的变量时, 推荐初始化的时候赋值 null, 因为这样在之后只需要检测这个变量的值是不是 null 就可以知道这个变量在后面是否被成功赋予了一个对象的引用;

Boolean 类型

Boolean 类型只有两个值, true 和 false

所有其他类型的值都可以通过 Boolean() 等价形式转换成 Boolean 类型;

数据类型 转换为 true 转换为 false
Boolean true false
String 非空字符串 “”空字符串
Number 非 0 数值 0, NaN
Object 任意对象 null
Undefined 不存在, 永远无法为 true undefined

Number 类型

es 中 Number 类型可以表示整数和浮点数

十进制, 八进制和十六进制

浮点数

在 es 中, 定义浮点数必须包含小数点, 而且小数点后面至少有一个数字; 虽然小数点前面可以不要求有整数, 但是推荐还是加上;

在小数点后面没有数字的情况下, 浮点数会直接被转换成整数; 在小数点后面是 0 的情况下, 也会直接转换成整数;

科学计数法

科学计数法的写法为 : 数值(整数或者浮点数) + e or E + 乘以 10 的多少次幂

浮点数精度

在 js 中, 浮点数精度运算非常诡异, 不要检测某个特定的浮点数值;

值的范围

最大值保存在 Number.MAX_VALUE 中, 最小值保存在 Number.MIN_VALUE 中;

如果超过了最大值则会自动转成正无穷 Infinity; 超过了最小值会转成负无穷 -Infinity;

正无穷和负无穷都不能再进行任何计算, 参与计算结果也是正无穷或者负无穷;

可以使用 isFinite()检测某个变量是否是有限数, 有限数返回 true, 无穷数返回 false

NaN

NaN 是一个特殊的数值, 表示的是不是数值; 用于表示本来要返回数值的操作, 但是失败了; 比如, 用 0 除任何数在其他语言中会直接导致错误, 但是在 ES 中, 0, +0, -0 或者 NaN 都有可能是返回值;

任何涉及到 NaN 的计算, 最终的结果都是 NaN

js 中提供了函数isNaN()来检测传入的参数是否是数值类型, 如果传入的参数可以转换成合法的数值, 那么返回 true, 否则返回 false;

Number()

Number()是转型函数, 可以将任意数据类型转成数值;

  1. 布尔值: true 转成 1, false 转成 0;

  2. 数值: 直接返回

  3. null: 返回 0

  4. undefined: 返回 NaN;

  5. 字符串

    1. 字符串中为合法数值, 包括字符前面的加减号, 直接转成含有符号十进制数值(+忽略);
    2. 字符串中以 0 开头的合法八进制, 直接忽略 0, 转成十进制数;
    3. 字符串中以 0x 和 0X 开头的合法十六进制数, 转成对应的十进制数;
    4. 字符串中以合法的浮点数, 则直接转换成对应的浮点数;
    5. 如果是空字符串, 则转为 0
    6. 如果字符串中的内容不是合法数值, 则返回 NaN;

  6. 对象

    1. 先调用valueOf()方法, 按照上面的规则进行转换, 如果转换的结果是 NaN, 则进行下一步参照 2

    2. 再调用toString()方法将对象转成字符串, 再按照字符串转换成 Number 的规则进行转换;

parseInt()

主要用于将字符串转成数值, 推荐优先使用该方法进行转换;

  1. 浮点数, 仅保留整数

  2. 字符串:

    1. 字符串前面的空格会被忽略, 从第一个非空格字符串开始转换;

    2. 如果第一个有效字符不是+,-或者有效数值, 那么会直接返回 NaN;

    3. 空字符进行转换的时候会直接返回 NaN;

    4. 如果开始是有效的数值或者+,-; 则会依次进行转换, 后面遇到无效字符则截断, 仅转换有效部分;

    5. 小数点也是无效字符, 所以转换的时候如果是浮点数, 则仅保留整数部分

    6. 如果字符串是0x开头, 则会被转换为十六进制;

    7. 如果字符是0开头, 且后面紧跟数值, 在非严格模式下会被某些实现解释成八进制整数;

  3. parseInt(param, index)支持第二个参数, 可以指定要转换数字的进制;

    1. 将十六进制转换为十进制:

    2. 将不同进制数字字符串转换为十进制

parseFloat()

parseFloat()函数的工作方式跟parseInt()类似;

  1. 从位置[0]开始检测每个字符;
  2. 依次对字符串进行解析, 直到末尾或者出现无效浮点数值为止, 一旦无效就返回当前有效结果;
  3. 第一次出现的小数点是有效的, 第二次出现的则无效, 一旦无效则返回当前有效结果;
  4. 始终忽略字符串开头的 0;
  5. 只能解析十进制, 如果入参是十六进制, 则直接返回 0;
  6. 如果字符串只有整数或者小数点后面只有 0, 那么parseFloat()会返回整数;

String 类型

String 字符串类型可以使用 0 个或者多个 16 位的 Unicode 字符序列; 字符串可以使用双引号, 单引号或者反引号`来表示;

字符字面量

字面量 含义
\n 换行
\t 制表
\b 退格
\r 回车
\f 换页
\\ 反斜杠
\‘ 单引号
\“ 双引号
\` 反引号
\xnn 十六进制编码 nn 的字符
\unnnn 十六进制编码nnnn 表示的 unicode 字符

字符串的长度

字符串可以通过length属性来获取长度

toString()

将其他数据类型转换为字符串;

  1. 几乎所有的 js 数据类型都有toString()方法, 唯一的用处就是将其他类型转换为字符串;

  2. toString()可以接受参数, 作用是将十进制转换为其他进制

String()

String()将其他类型转换为字符串;

  1. 如果传入的参数有 toString()方法, 则直接调用该方法;

  2. 如果传入的参数是 null, 则直接返回"null"

  3. 如果传入的参数是 undefined, 则直接返回 "undefined"

模板字面量

es6 中新增了模板字面量, 不同于单引号和双引号, 模板字面量可以换行定义字符串并保留换行符

字符串插值

在模板字面量中, 可以使用${}在字符串中插入变量, 也可以调用函数和方法; 只能用于使用了模板字面量的字符串中, 不能在””和’’定义的字符串中使用;

Symbol 类型

Symbol 是 ES6中新增的数据类型, 符号是原始值, 且符号实例是唯一的, 不可变的; 符号的用途是为了确保属性使用唯一标识, 不会发生属性冲突;

  1. 符号的基本使用

  2. 调用Symbol()函数时, 可以传入一个字符串作为参数对符号进行描述, 这个字符串参数与符号定义或者标识完全无关

  3. 只要创建 Symbol() 实例并将其作为对象的属性, 那么就可以保证它不会覆盖原有的属性, 无论是符号还是字符串属性;

  4. Symbol() 函数不能和 new 关键字一起作为构造函数使用, 因为 new 出来的是一个 Object, 而 Symbol()的话需要产生一个 symbol 的基础数据类型

  5. 全局符号注册表: 通过 Symbol.for()来注册全局符号, 方便符号的复用; 调用Symbol.for()的时候, 会先检查全局注册表中是否存在对应的符号, 如果存在那么就会调用这个存在的符号进行操作, 如果不存在, 那么就会新建一个符号并返回符号实例;

  6. 采用相同的符号描述, 在全局注册表中定义的符号和使用 Symbol.for()定义的符号也并不等同;

  7. 全局注册表中的符号必须使用字符串键来创建, 因此作为参数传给Symbol.for()的任何值都会被转换成字符串;

  8. 全局注册表中可以使用 Symbol.keyFor()来查询全局注册表, 这个方法接受符号, 返回该全局符号对应的字符串键, 如果查询的不是全局符号, 则返回 undefined;

  9. 使用符号作为属性

Object 类型

  1. es 中的对象其实就是一组数据和功能的集合;
  2. 对象通过 new 操作符和对象类型的名称来创建, 比如 let obj = new Object();
  3. Object也是派生其他类型对象的基类, Object 类型的所有属性和方法在派生对象的身上也同样存在;

Object 属性和方法

  1. constructor: 用于创建当前对象的函数, 例如在let obj = new Object();中, Object()函数就是constructor;
  2. hasOwnProperty(propertyName): 用于判断当前对象实例(不是原型)上是否存在给定的属性, 要检查的属性名必须是字符串或者符号;
  3. isPrototypeof(object): 判断当前调用isPrototypeof()方法的对象是否是入参 object 的原型
  4. propertyIsEnumerable(propertyName): 判断入参属性对用的值是否可以使用for-in语句进行枚举
  5. toLocalString(): 返回对象的字符串表示, 该字符串反应了对象所在本地执行环境;
  6. toString(): 返回对象的等价字符串表示;
  7. valueOf(): 返回对象的字符串, 数值或者布尔值表示; 通常和toString()返回的结果相同;
